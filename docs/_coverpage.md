# Requests-HTML

> HTML Parsing for Humans (writing Python 3)!

*   全面支持解析JavaScript!
*   CSS 选择器 (jQuery风格, 感谢PyQuery).
*   XPath 选择器, for the faint at heart.
*   自定义user-agent (就像一个真正的web浏览器).
*   自动追踪重定向.
*   连接池与cookie持久化.
*   令人欣喜的请求体验，魔法般的解析页面.
